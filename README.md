# NumSim


## Build the code
### Release
```shell
mkdir build && cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make -j
```

### Debug

```shell
mkdir build && cd build
cmake -DCMAKE_BUILD_TYPE=Debug ..
make -j
```
